# Competitive Programming

This repository contains selected solutions to algorithmic problems from competitive programming platforms, primarily **Codeforces**.

The goal of this repository is to document interesting problems and their solutions, focusing on different algorithmic techniques such as combinatorics, graph algorithms, greedy methods, string hashing, and number theory.

---

## Topics Covered

- Combinatorics
- Number Theory
- Graph Algorithms
- Tree Algorithms
- Greedy Algorithms
- String Hashing
- Dynamic Programming

---

## Codeforces Problems

| Problem | Main Topic |
|-------|------|
| Baby Ehab Plays with Permutations | Combinatorics / DP |
| Bessie's Birthday Cake (Hard) | Greedy / Geometry |
| Games on a CD | String Hashing |
| The Number of Pairs | Number Theory |
| Xor-tree | Tree DFS |
